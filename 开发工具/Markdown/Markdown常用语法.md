# Markdown 常用语法

## 标题

```
# 一级标题
## 二级标题
### 三级标题
```

## 换行

方式一：`<br>`。

第一行<br>第二行

方式二：两个空格。

第一行  
第二行

## 引用

> 1
> 2
> 3

## 复选框

- [ ] task1
- [x] task2

## 表格

| 1    |  2   |    3 |
| ---- | :--: | ---: |
| A    |  B   |    C |

## 脚注

脚注[^footnote]。

## 分割线

---

## 生成目录

[toc]

> `[toc]`：table of content。
